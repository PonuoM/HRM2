# Database Migrations

This directory will contain SQL migration files for setting up the database schema.

The migrations will be created in the next task (Task 2).

## Structure

- `001_create_users_table.sql` - Create users table
- `002_create_time_logs_table.sql` - Create time_logs table  
- `003_create_leave_types_table.sql` - Create leave_types table
- `004_create_leave_balances_table.sql` - Create leave_balances table
- `005_create_leave_requests_table.sql` - Create leave_requests table
- `006_insert_default_data.sql` - Insert default leave types and sample data
